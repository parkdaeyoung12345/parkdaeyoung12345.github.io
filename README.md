# 🎨 대영이의 AI Jelly 놀이터  
**AI × 저학년 학습 × 게임형 인터랙션**  
말랑말랑한 젤리 UI 속에서 AI가 어떻게 배우는지를 놀이로 체험하는 웹페이지입니다.

---

## 🌟 프로젝트 소개
이 웹사이트는 **초등 저학년 학생들이 AI 개념을 쉽고 재미있게 이해하도록**  
게임 기반으로 설계된 **AI 융합교육 콘텐츠**입니다.

AI가 실제로 데이터를 보고 학습하는 흐름을  
① 도형 분류  
② 퍼즐 특징 추출  
③ 시간 예측  
④ 패턴 추론  

네 가지 게임을 통해 자연스럽게 접할 수 있습니다.

---

## 🧩 제공되는 AI 학습 게임

| 게임명 | 설명 | 관련 AI 개념 |
|--------|------|---------------|
| 🔵 **도형 젤리 찾기** | 도형을 선택하며 규칙을 찾는 놀이 | 이미지 분류(Image Classification) |
| 🧩 **젤리 퍼즐 맞추기** | 조각난 그림의 특징을 AI처럼 찾기 | 특징 추출(Feature Extraction) |
| ⏰ **젤리 시계 맞추기** | 다음 시간을 예측하는 활동 | 시계열 예측(Time Series Prediction) |
| 🎨 **무늬 젤리 예측하기** | 반복 패턴을 이해하고 예측하기 | 패턴 인식(Pattern Recognition) |

각 게임의 최고 기록은 자동 저장되며,  
메인 페이지에서 **총 점수**로 통합 확인할 수 있습니다.

---

## 💡 왜 이런 게임을 만들었나요?
AI는 **데이터 속 규칙을 배우는 과정**을 반복합니다.  
아이들이 이 개념을 몸으로 체감하도록 하기 위해  
게임 속에서 다음과 같은 학습 흐름을 직접 경험하도록 구성했습니다:

- 관찰하기 → 규칙 찾기 → 예측하기 → 평가 받기  
- 잘하면 AI의 “이해도(학습 게이지)”가 올라가도록 시각화  
- 반복할수록 AI 이해에 대한 메타학습(meta-learning) 효과 증진

## 🎨 디자인 컨셉
이 프로젝트는 **Liquid Glass + Jelly UI** 스타일을 적용했습니다.

- 말랑한 인터랙션 → 초등 저학년에게 친숙  
- 투명한 글래스 카드 → AI 정보가 시각적으로 깔끔하게 전달  
- 은은한 블롭 애니메이션 → AI의 사고 흐름을 비유적으로 표현
